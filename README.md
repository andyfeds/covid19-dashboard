# covid19-dashboard
## A dashboard built using Vue.js to track latest COVID-19 updates in India.

## Version
1.0.0

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
